# Algorhythm

AI-powered practice platform for algorithmic programming problems.

What it does

- Lets users register, log in, and track sessions.
- Shows algorithmic programming questions with problem statements, starter code and topics.
- Provides an in-browser code editor and language selection.
- Allows running code (Judge0) and submitting solutions validated against stored test cases.
- Offers an LLM-powered chat assistant for hints and explanations.

Main technologies

- Backend: Kotlin + Spring Boot
- Frontend: React
- Database: PostgreSQL
- Code execution: Judge0
- LLMs / AI integrations: Gemini, Groq (configurable via API keys)
- Build: Gradle
- Optional: Docker & docker-compose
- Deployment: Deployed on an Oracle VM
